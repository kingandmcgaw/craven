File structure

This is an outline of the CSS file structure. Currently this is still a work-in-progress and therefore subject to change. The general idea is that all modules have their own .scss file, all of which are imported into main.scss, which compiles to one compressed, CSS file in production.

  • main.scss
    • variables.scss
    • mixins.scss
    • reset.scss
    • typography.scss
    • tables.scss
    • forms.scss
    • layout.scss
    • header.scss
    • navigation.scss

HTML template

This is an example of the skeleton HTML template required for each page. In production this may differ as includes will serve assets such as stylesheets and JavaScript. It has been placed in a textarea to make copying and pasting of this template trivial.
