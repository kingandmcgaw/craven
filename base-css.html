Icons

The CSS for the icons is taken from Bootstrap, and the icons themselves are provided by Glyphicons.

Icons can be included by adding an empty i tag with the relevant class for the icon you wish to display. To display the white version of an icon, append the class .icon-white. For example:

<!-- Displays glass icon -->
<i class="icon-glass"></i>

<!-- Displays white glass icon -->
<i class="icon-glass icon-white"></i>
