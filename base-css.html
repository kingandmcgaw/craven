Typography

The typography in this pattern library is built using ems, meaning the entire typographic scale can be altered by adjusting the $base-font-size variable.

Links

Links in this pattern library are fairly standard. There are two variables which can be altered to change the colour of the links: $link-color and $link-color-hover.
